Project Description

The Visualization Tool for Electric Vehicle Charge and Range Analysis is a data visualization project developed using Tableau to analyze electric vehicle data. The project provides interactive dashboards and stories that help users understand charging stations, battery range, vehicle prices, top speed, and brand comparisons through meaningful visualizations.

Objectives

- Develop an interactive Tableau dashboard and story.
- Analyze EV charging stations and infrastructure.
- Compare electric vehicle brands, price, and range.
- Present data in an easy-to-understand visual format.

Technologies Used

- Tableau Public
- Microsoft Excel
- CSV Datasets
- GitHub

Features

- Interactive Dashboard
- Interactive Story
- EV Brand Analysis
- Charging Station Analysis
- Price Comparison
- Battery Range Analysis
- Top Speed Analysis
