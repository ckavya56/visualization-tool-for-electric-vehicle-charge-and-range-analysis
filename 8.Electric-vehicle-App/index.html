About the Project

EV Charge & Range Analytics is an interactive data visualization project developed to analyze the growing Electric Vehicle ecosystem using Tableau dashboards and stories. The project provides valuable insights into EV charging infrastructure, vehicle range, pricing, and brand performance across global and Indian markets.

  • Interactive Tableau Dashboard for EV analytics.
  • Visualization of charging stations, vehicle range, price, and brand distribution.
  • Data collected from multiple datasets and analyzed using MySQL and Tableau.

This project combines multiple electric vehicle datasets to generate meaningful business insights through interactive visualizations. Bootstrap is used for designing the web interface, MySQL for managing the datasets, and Tableau Public for publishing dashboards and stories, making the analysis easy to understand and explore.
